ADB-PRO

представляю вам мою сборку ADB (ADB-PRO)

В в ней собрано очень много ультилит/программ для ADB/ANDROID

в сборку входит: 

драйвера всех/для всех устройств android/adb/fastboot/port/mtp/internet-controller

jadx - программа для редактирования *.apk *.lib все виды пакетов написаных на/для JAVA

MIT - программа для изменения разделов прошифки android (system.img, и д.р.)

MyPhoneExplorer - программа для управления android 0.0.1-8.8.8 (в основном для старых android <7)

scrcpy - программа для вывода изображения с устройства android >4.4.4

ADB/fastboot/shell

TFTUnlock (MOD) — программа для ремонта смартфонов на базе Android. Она помогает устранять различные проблемы с устройствами, включая снятие блокировок экрана, обход защиты Google и др.

Factory Reset (FRP), восстановление повреждённых номеров IMEI, разблокировку загрузчиков и другие функции. подробнее github

ADBAppControl (MOD) - позволяет управлять приложениями на устройствах под управлением Android через удобный графический интерфейс. ( возможности ADB AppControl: отключение и удаление приложений без root-прав; установка пользовательских приложений; сохранение apk-файлов установленных приложений; менеджер разрешений для приложений; сохранение и загрузка списков-пресетов приложений; быстрый поиск приложений в Google Play, ApkMirror, F-Droid и других; отображение информации об устройстве; изменение разрешения экрана; скрытие значков статус-бара; возможность удалённого управления устройством; виртуальные кнопки громкости, питания, камеры и навигации; создание скриншотов экрана устройства; перезагрузка в recovery и bootloader. и д.р )

собраны почти все команды adb/fastboot/shell (команды находятся в филе ADB.txt)

ResearchDownload - программа для прошивки смартфонов и планшетов на базе процессоров SpreadTrum и не только

SP Flash Tool - Позволяет загружать и обновлять прошивки, форматировать разделы памяти, восстанавливать устройства после неудачных обновлений и д.р

И МНОГИЕ ДРУГИЕ.
